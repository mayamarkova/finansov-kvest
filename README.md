# 🎓 Финансов Куест

Интерактивна игра за финансова грамотност, създадена за **Деня на отворените врати** на **Русенски университет** – специалност **Финансова математика**.

## 🎮 За играта

Играта съдържа **5 нива** с **25 въпроса** по теми от финансовата грамотност:

| Ниво | Тема | Точки за верен | Наказание за грешен |
|------|------|:--------------:|:-------------------:|
| 1 | 💰 Лични финанси | +10 | -3 |
| 2 | 🏦 Банково дело | +20 | -5 |
| 3 | 📈 Инвестиции | +30 | -7 |
| 4 | 🔢 Лихви и изчисления | +40 | -10 |
| 5 | 📋 Данъци и финанси | +50 | -12 |

### Правила:
- Нужни са минимум **3 от 5** верни отговора за преминаване на следващо ниво
- При грешен отговор се губят точки (по-малко от наградата за верен)
- След всеки въпрос има **обяснение** на правилния отговор
- **Класацията** запазва резултатите на всички играчи

## 🚀 Как да го пусна?

### Вариант 1: GitHub Pages (препоръчително)

1. Създай ново репо в GitHub
2. Качи файловете:
   ```bash
   git init
   git add .
   git commit -m "Финансов Куест"
   git branch -M main
   git remote add origin https://github.com/ТВОЕТО-ИМЕ/finansov-kvest.git
   git push -u origin main
   ```
3. Отиди в **Settings → Pages → Source → GitHub Actions**
4. Изчакай 1-2 минути – готово! 🎉

Играта ще е на: `https://ТВОЕТО-ИМЕ.github.io/finansov-kvest/`

### Вариант 2: Локално

```bash
npm install
npm run build
```

Отвори `dist/index.html` в браузър.

## 📱 Мобилно-ориентиран дизайн

Играта е оптимизирана за използване на телефон – перфектна за игра "на крак" на деня на отворените врати!
